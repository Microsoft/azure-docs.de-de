# <a name="manage-your-azure-marketplace-and-appsource-publisher-profile"></a>Verwalten Ihres Azure Marketplace- und AppSource-Herausgeberprofils

In diesem Artikel wird erläutert, wie Sie im [Cloud-Partnerportal](https://cloudpartner.azure.com/) Ihr Herausgeberprofil anzeigen und verwalten.  Für den Zugriff auf dieses Portal müssen Sie bereits als Microsoft Cloud-Partner registriert sein.  Weitere Informationen finden Sie unter [Erste Schritte mit dem Cloud-Partnerportal](../../cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal.md).

Es wird davon ausgegangen, dass Sie bereits die erforderlichen Schritte ausgeführt haben, um Azure Marketplace- und AppSource-Herausgeber zu werden. Das Herausgeberprofil wird, nachdem Ihr Partnerantrag genehmigt wurde, im Cloud-Partnerportal registriert. Ihr Herausgeberprofil gilt für alle Angebote und SKUs, die über das Konto veröffentlicht werden, das bei der Partnerregistrierung verwendet wurde.


## <a name="publisher-profile"></a>Herausgeberprofil

Ihr Herausgeberprofil charakterisiert Ihr Unternehmen im Azure Marketplace und in AppSource. Es besteht aus Herausgeber-ID, Anzeigename und Besitzer-E-Mail-Adressen. Ein gut verwaltetes Profil erhöht Ihre Sichtbarkeit und ermöglicht Benutzern, entsprechende Angebote im Marketplace einfach zu identifizieren und auszuwählen.

> [!NOTE]
> Ihre Herausgeber-ID und die E-Mail-Adresse des registrierten Besitzers können nach dem Veröffentlichen des ersten Angebots nicht mehr geändert werden. Im Cloud-Partnerportal können Sie jedoch Ihr Profil anzeigen und den Anzeigenamen des Herausgebers bearbeiten.  <br/>
> Im Abschnitt „Benutzer“ im linken Navigationsmenü im Cloud-Partnerportal können Sie Ihrem Veröffentlichungsprofil Benutzer (Mitwirkende und Besitzer) hinzufügen.


### <a name="view-and-manage-your-publisher-profile"></a>Anzeigen und Verwalten Ihres Herausgeberprofils

Zum Anzeigen und Verwalten Ihres Herausgeberprofils klicken Sie auf der Menüleiste oben rechts auf die Registerkarte **Publisher profile** (Herausgeberprofil).

![Registerkarte „Herausgeberprofil“ – Herausgeberprofil](./media/publisherprofilenew.png)

Das Dialogfeld für Ihr **Profil** wird angezeigt.

![Details eines Herausgeberprofil – Herausgeberprofildetails](./media/publisherprofiledetails.png)


### <a name="link-your-dev-center-account"></a>Verbinden Ihres Dev Center-Kontos

Sie können auch Ihr vorhandenes Dev Center-Konto mit Ihrem Herausgeberprofil im Cloud-Partnerportal verknüpfen.  Melden Sie sich dazu zunächst beim Portal mit der gleichen E-Mail-Adresse an, die Sie für die Registrierung Ihres Dev Center-Kontos verwendet haben. Daraufhin werden auf der Seite mit Ihrem Herausgeberprofil der Status Ihres Dev Center-Kontos, die E-Mail-Adresse des Dev Center-Kontobesitzers und der Dev Center-Kontoname angezeigt.

> [!NOTE]
> Die Registrierung des Developer Center-Kontos ist für die Veröffentlichung kostenpflichtiger SKUs im Marketplace obligatorisch.

Wenn Sie nicht über ein Developer Center-Konto verfügen, können Sie wie unter [Registrieren bei Dev Center](https://docs.microsoft.com/azure/marketplace/register-dev-center) beschrieben ein Konto erstellen.  Fügen Sie dann die registrierte E-Mail-Adresse als Besitzer im Cloud-Partnerportal für Ihr Angebot hinzu. Ein Besitzer kann im Abschnitt **Benutzer** Ihrem Veröffentlichungsprofil nur die registrierte E-Mail-Adresse hinzufügen. Melden Sie sich dann beim Portal mit derselben E-Mail-Adresse an, und stellen Sie eine Verbindung mit Ihrem Dev Center-Konto her.

![Details eines Herausgeberprofil – Herausgeberprofilbeschreibung](./media/publisherprofiledescription.png)
