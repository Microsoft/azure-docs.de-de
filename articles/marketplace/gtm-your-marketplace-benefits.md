# <a name="your-marketplace-benefits"></a>Ihre Marketplace-Vorteile

Im Rahmen Ihrer Veröffentlichungsaktivitäten müssen Sie technische und marketingbezogene Objekte für Ihr Angebot anlegen. Infolgedessen gibt es technische und marketingbezogene Vorteile, die Azure Marketplace- und AppSource-Herausgeber nutzen können, um die Kundenbindung an Ihr Angebot zu verbessern. Die folgenden technischen Ressourcen können für Sie nützlich sein.

## <a name="technical-resources-provided-by-microsoft"></a>Technische Ressourcen, die von Microsoft bereitgestellt werden

Während Sie Ihr Angebot erstellen, können Sie eine Reihe von Microsoft-Ressourcen nutzen, z. B. die Beratung durch einen technischen Experten oder die Unterstützung beim Testen einer Marketplace-spezifischen Lösung.

Technische Ressourcen können jederzeit genutzt werden.  In der folgenden Tabelle sind einige dieser Vorteile aufgeführt:

![](./media/marketplace-publishers-guide/technical-benefit-table.png)

## <a name="how-to-access-microsoft-resources"></a>Zugriff auf Microsoft-Ressourcen

Die Vorteile bei der Markteinführung sind für Partner zugänglich, die kürzlich ein Angebot veröffentlicht haben. 

1. Onboarding: [Registrieren Sie sich](https://azuremarketplace.microsoft.com/sell), um Zugriff auf das Cloud-Partnerportal und die Onboarding-Supportressourcen zu erhalten.
2. Microsoft Marketplace-Beratung: Nutzen Sie die Vorteile dieser [KOSTENLOSEN Beratung](https://support.microsoft.com/help/4010317/microsoft-marketplaces-consultation) zum Planen die Architektur Ihrer App im Marketplace.
3. Marketplace-Foren: Stellen Sie Ihre Fragen direkt an Microsoft-Mitarbeiter und andere Marketplace-Partner im [Marketplace-Forum](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222).
4. Technischer Support: Verwenden Sie das Menü „Hilfe“ im [Cloud-Partnerportal](https://cloudpartner.azure.com/), um Antworten auf Ihre Supportfragen zum Marketplace zu erhalten. 
5. Sponsorship für AAD-Integration, Entwicklung und Test: Sie werden registriert, wenn Sie ein transaktionsfähiges (bereitstellbares und abrechenbares Angebot) live schalten. Im nächsten Abschnitt finden Sie Informationen dazu, wie Sie mit den Vorteile bei der Markteinführung beginnen können.

## <a name="go-to-market-gtm-benefits-in-the-marketplace"></a>Vorteile bei der Markteinführung im Marketplace

Neue Listungen, sowohl im Azure Marketplace als auch in AppSource, sind berechtigt, über unser Microsoft GTM Services-Team kostenlose Marketplace-GTM-Vorteile zu erhalten. Dieses Team hilft Ihnen, Ihre Listungen zu optimieren und die Bekanntheit Ihrer Lösungen im Microsoft Marketplace zu erhöhen.

| Storefront | Zusätzliche verfügbare Vorteile |
|:--- |:--- |
| AppSource |  Microsoft Gold-Partner <br> Transaktionsangebote – Marketplace-Vorteile <br> Test-Apps |
| Der Azure Marketplace | Transaktionsangebote – Marketplace-Vorteile <br> Testversionsangebote |

Die Unterstützung bei der Markteinführung umfasst z.B. Vorlagen, Webinhalte, Schulungen und Tools, um den Bekanntheitsgrad Ihres Unternehmens zu fördern.

Um von diesen Vorteilen zu profitieren, müssen Sie Folgendes tun:

1. Veröffentlichen Sie ein Angebot entweder im Azure Marketplace oder in AppSource.
2. Stellen Sie sicher, dass Sie im Abschnitt **Kontaktinformationen** Ihres Angebots einen Marketingkontakt eingegeben haben. Dies sollte eine dedizierte Marketingressource sein und kein Catch-All-Alias. (vermeiden Sie deshalb „support\@“, „info\@“, „marketing\@“ usw.)

Basierend auf Ihrem Lösungsstatus, je nachdem, ob das Angebot List, Trial oder Transact – Marketplace-Vorteile – ist, erhalten Sie entweder eine E-Mail mit Selbsthilferessourcen oder eine Einladung zu einem Beratungsgespräch mit einem dedizierten Kundenbindungsmanager. Sie müssen nichts tun, um den Anruf einzuleiten – das Marketplace Onboarding-Team wird sich auf der Grundlage der Informationen, die Sie im [Cloud-Partnerportal](https://cloudpartner.azure.com/) angeben, an Sie wenden.

Der Umfang der für Sie verfügbaren Aktivitäten erweitert sich mit der Anzahl Ihrer Angebote auf dem Markt. Allen Listungen steht eine Basisauswahl an Optimierungsempfehlungen und Höherstufungen zur Verfügung.  Diese Listungen sind auch für zusätzliche Marketingvorteile qualifiziert, die auf dem Lösungsstatus (List, Trial oder Transact – Marketplace-Vorteile), dem gewählten Marketplace und dem erreichten Microsoft Partner-Kompetenzerfolg basieren.
