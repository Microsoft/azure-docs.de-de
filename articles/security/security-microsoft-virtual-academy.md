# <a name="azure-security-courses-from-microsoft-virtual-academy"></a>Azure-Sicherheitskurse der Microsoft Virtual Academy
In der Microsoft Virtual Academy werden kostenlose Onlineschulungen angeboten, damit Entwickler, IT- und Datenspezialisten und Studenten immer auf dem neuesten Stand der Technologie bleiben, ihre Kompetenzen und Kenntnisse vertiefen und sich weiterentwickeln können.

Auf dieser Seite finden Sie eine Aufstellung ausgewählter Kurse zur Azure-Sicherheit. Besuchen Sie die [Microsoft Virtual Academy](https://mva.microsoft.com/) , um sich einen Überblick über alle verfügbaren Kurse zu verschaffen.

[Dev/Test in the Cloud (Entwicklung und Testen in der Cloud)](https://mva.microsoft.com/en-us/training-courses/devtest-in-the-cloud-16274?l=9aAijd7LC_2005190311)

Möchten Sie als Entwickler schneller bessere Anwendungen bereitstellen? Dies können Sie durch Verschieben Ihrer Entwicklungs- und Testumgebungen in die Cloud erreichen. Erfahren Sie, wie Sie dabei vorgehen müssen, und lernen Sie die Vorteile dieses Wechsels kennen. In Demos und Präsentationen wird außerdem gezeigt, wie Ihre Anforderungen an die Entwicklungs- und Testumgebungen mit Microsoft Azure unterstützt werden können. Die Schulung umfasst Lektionen zu Methoden der Sicherheitsentwicklung und Bereitstellung.

[Common Tasks for Linux on Azure (Häufige Aufgaben für Linux in Azure)](https://mva.microsoft.com/en-us/training-courses/common-tasks-for-linux-on-azure-16191?l=J0Hvb7qJC_1204668937)

In diesem ausführlichen Kurs erhalten Sie Antworten auf Ihre Fragen zur Verwendung von Linux auf der Microsoft Azure Platform. Lernen Sie mit den Experten der [Linux Academy](https://linuxacademy.com/)einige allgemeine Aufgaben kennen. Sie erhalten Informationen zum Erstellen eines virtuellen Linux-Computers in Azure, zum Zugreifen auf virtuelle Linux-Computer über Remotedesktop-Software und zum Ausführen von virtuellen Hosts. In diesem Kurs werden viele Sicherheitstechnologien und -konfigurationen behandelt.

[Secure the Cloud (Sichern der Cloud)](https://mva.microsoft.com/en-us/training-courses/secure-the-cloud-14037?l=lQIkkst0B_5300115881)

In dieser Sitzung erfahren Sie, wie Microsoft Sie bei der Erfüllung globaler Compliancevorgaben, z.B. ISO 27001/27018, FedRAMP, PCI und HIPAA, mit neuen Sicherheitskontrollen unterstützen kann. Diese Kontrollelemente umfassen beispielsweise Folgendes: Verschlüsselung für ruhende Daten, Schlüsselverwaltung, Schutz für virtuelle Computer, Protokollierung und Überwachung, Antischadsoftwaredienste, Identitätsverwaltung, Zugriffssteuerungen und vieles mehr.

[Design and Implement Cloud Data Platform Solutions (Entwickeln und Implementieren von Lösungen für Datenplattformen in der Cloud)](https://mva.microsoft.com/en-us/training-courses/design-and-implement-cloud-data-platform-solutions-15711?l=jbCdW0j1B_3005244527)

Lernen Sie die Features und Funktionen der Lösungen für Datenplattformen in der Microsoft-Cloud kennen. Verschaffen Sie sich einen allgemeinen Überblick über Plattformen, und informieren Sie sich über Sicherheitsfeatures, Optionen für Hochverfügbarkeit, Verfahren für die Überwachung und Verwaltung von Clouddaten und andere Dinge. Darüber hinaus erhalten Sie Anleitungen zum Identifizieren von Kompromissen und zur Entscheidungsfindung beim Entwerfen von öffentlichen und Hybrid Cloud-Lösungen mithilfe der Features für Datenplattformen in der Microsoft-Cloud.

[Manage and Secure Identities in a Cloud and Mobile World (Verwalten und Sichern von Identitäten in der Cloud und der mobilen Welt)](https://mva.microsoft.com/en-us/training-courses/manage-and-secure-identities-in-a-cloud-and-mobile-world-14013?l=GIJ2GcvrB_405192797)

In dieser Sitzung erfahren Sie, wie Sie mithilfe von Azure Active Directory und Microsoft Advanced Threat Analytics die Identitäten der Benutzer schützen und verwalten, wie Sie Sicherheitsverletzungen erkennen können, bevor sie Schaden anrichten, und wie Sie für Ihre Benutzer eine einzelne Identität für den Zugriff auf alle Unternehmensressourcen bereitstellen. Lernen Sie die Technologien kennen, mit denen Sie Schatten-IT ermitteln, den Anwendungszugriff verwalten und verdächtige Aktivitäten über erweiterte Sicherheitsberichte und Analysen, Überwachung und Warnungen hinsichtlich des Benutzerverhaltens überwachen können.

[Security in a Cloud-Enabled World (Sicherheit in einer Welt mit Cloudumgebungen)](https://mva.microsoft.com/en-us/training-courses/security-in-a-cloudenabled-world-12725?l=CfLHobAcB_3904300474)

Experten erläutern Ihnen anhand des Posters [Microsoft Cloud Security for Enterprise Architects](https://www.microsoft.com/download/48121) (Microsoft-Cloudsicherheit für Enterprise-Architekten) die Verantwortungsbereiche des Kunden. Sie geben auch Tipps zur Erneuerung aller Bereiche Ihres Sicherheitsstatus, einschließlich Governance, Einschlussstrategien, Sicherheitsvorgängen, professionellem Assetschutz, Information Protection sowie Benutzer- und Gerätesicherheit mit Schwerpunkt auf dem Schutz von Administratorberechtigungen. Sie machen sich mit dem Framework vertraut, das auch von den Microsoft-Mitarbeitern für die Internetsicherheit verwendet wird, um die Cloudsicherheit von Kunden zu bewerten und für sie eine Sicherheitsroadmap zu erstellen.

[Microsoft Azure IaaS Deep Dive (Tiefgehende Informationen zu Microsoft Azure IaaS)](https://mva.microsoft.com/en-us/training-courses/microsoft-azure-iaas-deep-dive-14339?l=PtppYVQgB_8300115888)

Erfahren Sie, wie Sie die Infrastrukturfunktionen von Microsoft Azure nutzen können. Als IT-Experte müssen Sie über keinerlei Vorkenntnisse in Azure verfügen. In diesem Kurs wird schrittweise die Erstellung und Konfiguration von virtuellen Azure-Computern, virtuellen Azure-Netzwerken und der standortübergreifenden Konnektivität zum Aufnehmen des Betriebs in der Cloud erläutert. Auch auf Sicherheitsfeatures und-überlegungen wird während des gesamten Kurses eingegangen.

[Getting Started with Azure Security for the IT Professional](https://mva.microsoft.com/training-courses/getting-started-with-azure-security-for-the-it-professional-11165?l=HfHzCXSAB_7404300474)

In dieser Schulung erläutert ein Team aus Sicherheitsexperten und Azure-Technikern anhand zahlreicher Demos, welche Möglichkeiten es in Azure gibt, die über die grundlegenden Zertifizierungen hinausgehen. Hier erhalten Sie Informationen zum Entwurf und zur Verwendung verschiedener Technologien, um sicherzustellen, dass Sie über die Sicherheit und Architektur verfügen, die für den erfolgreichen Start Ihrer Projekte in der Cloud erforderlich sind. Verschaffen Sie sich einen gründlichen Einblick in die Abläufe im Datencenter, die Konfiguration von virtuellen Computern, die Netzwerkarchitektur und die Speicherinfrastruktur.

[Deep Dive into Azure Resource Manager Scenarios and Patterns (Tiefgehende Informationen zu Azure Resource Manager-Szenarien und -Mustern)](https://mva.microsoft.com/en-us/training-courses/deep-dive-into-azure-resource-manager-scenarios-and-patterns-13793?l=i1m06ZJYB_7001937557)

Lernen Sie Azure Resource Manager mit einem Team aus internationalen Experten kennen, die Ihnen Skripts und Tools vorstellen, mit denen Sie mühelos Elemente Ihrer Anwendungsinfrastruktur verschieben können. Machen Sie sich mit der Verwendung der rollenbasierten Zugriffssteuerung (RBAC) zum Implementieren der Sicherheit mit Azure Resource Manager vertraut.

[Azure Rights Management Services Core Skills (Wichtige Fähigkeiten für Azure Rights Management-Dienste)](https://mva.microsoft.com/en-us/training-courses/azure-rights-management-services-core-skills-10500?l=QLoxMwuCB_1805094681)

In diesem Kurs wird erläutert, weshalb Information Protection ein Muss in Ihrer Organisation ist und wie das geistige Eigentum Ihrer Organisation auf allen Geräten und in der Cloud durch die Rechteverwaltung geschützt wird. Microsoft-Experten vermitteln Ihnen entsprechende praktische Kenntnisse und technisches Know-how.
