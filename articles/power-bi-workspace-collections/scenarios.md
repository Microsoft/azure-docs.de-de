# <a name="common-power-bi-workspace-collection-scenarios"></a>Häufige Szenarien für Power BI-Arbeitsbereichssammlungen

Power BI-Arbeitsbereichssammlungen zielen auf das Verfügbarmachen der meisten vorhandenen Power BI-API-Funktionen als Teil eines Azure-Diensts ab, mit dem Sie Anwendungen entwickeln können.  Darüber hinaus können Sie die erforderlichen Ressourcen und den Power BI-Inhalt programmgesteuert bereitstellen, entwickeln und bereitstellen. Weitere Informationen finden Sie unter [Erste Schritte mit Microsoft Power BI-Arbeitsbereichssammlungen](get-started.md).

> [!IMPORTANT]
> Power BI-Arbeitsbereichssammlungen sind veraltet und nur noch bis Juni 2018 (oder bis zum Termin in Ihrem Vertrag) verfügbar. Es wird empfohlen, die Migration zu Power BI Embedded zu planen, um Unterbrechungen für Ihre Anwendung zu vermeiden. Informationen zur Migration Ihrer Daten zu Power BI Embedded finden Sie unter [Migrieren von Inhalten aus Power BI-Arbeitsbereichssammlungen zu Power BI Embedded](https://powerbi.microsoft.com/documentation/powerbi-developer-migrate-from-powerbi-embedded/).

Nachstehend finden Sie einige Szenarien für die Verwendung von **Power BI-Arbeitsbereichssammlungen** in Ihren Anwendungen.

* Vollständig interaktive Berichte, die auf dem Power BI Desktop erstellt wurden, können mithilfe eines neuen Anwendungstoken-Authentifizierungsmodells in Ihre eigenen Anwendung eingebettet werden.
* Unabhängige Softwarehersteller und Kunden, die kundenorientierte Anwendungen erstellen, können den Dienst **Power BI-Arbeitsbereichssammlung** und das Power BI SDK verwenden, um interaktive Berichte einzubetten. Informationen finden Sie unter [Erste Schritte mit Microsoft Power BI-Arbeitsbereichssammlungen](get-started.md).
* Als Entwickler können Sie das Power BI-Visualisierungsframework zum Erstellen von benutzerdefinierten Visualisierungen verwenden, die in Ihrer eigenen App verwendet werden können. Weitere Informationen finden Sie unter [Erweitern Sie Power BI um Ihre eigenen Visualisierungen](https://powerbi.microsoft.com/custom-visuals/).
