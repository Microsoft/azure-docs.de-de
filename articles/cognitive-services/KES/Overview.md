# <a name="knowledge-exploration-service"></a>Knowledge Exploration Service

Willkommen beim Microsoft Knowledge Exploration Service.  Knowledge Exploration Service (KES) bietet eine schnelle und effektive Möglichkeit, Ihre Anwendungen mit einer interaktiven Suche und Verfeinerungen zu ergänzen.  Mit KES können Sie einen komprimierten Index aus strukturierten Daten zusammenstellen, eine Grammatik erstellen, die Abfragen in natürlicher Sprache interpretiert, und interaktive Abfrageformulierungen mit Vorschlägen zur automatischen Vervollständigung anbieten.

Unter [Erste Schritte](GettingStarted.md) finden Sie eine exemplarische Vorgehensweise, wie Sie mit KES eine interaktive Suchoberfläche für eine akademische Publikationsdomäne erstellen können.

![AcademicAutoComplete](AutoComplete.png)

KES wurde von Cognitive Services Preview zu Cognitive Services Labs verschoben. Die neue Startseite für das Projekt lautet [ https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration ](https://labs.cognitive.microsoft.com/en-us/project-knowledge-exploration).
