Das Azure Spatial Anchors SDK sollte mit den folgenden Paketen in <a href="https://unity.com/releases/2019-lts" target="_blank">Unity 2019.4 (LTS)</a> konfiguriert werden:
- AR Foundation: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Windows Mixed Reality: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- ARCore XR-Plug-In (für Unterstützung von Android-Geräten): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- ARKit XR-Plug-In (für Unterstützung von iOS-Geräten): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> Das ASA SDK unterstützt das <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">Unity XR-Plug-In-Framework</a> zurzeit nicht.
